# 基于NLP的搜索引擎用户画像项目

## 项目简介
本项目旨在利用自然语言处理和机器学习技术，基于用户历史查询词和人口属性标签，构建用户画像并预测新增用户的人口属性（年龄、性别、学历）。

## 数据说明
- **ID**：加密后的用户ID
- **Age**：年龄（0：未知；1：0-18岁；2：19-23岁；3：24-30岁；4：31-40岁；5：41-50岁；6：51-999岁）
- **Gender**：性别（0：未知；1：男性；2：女性）
- **Education**：学历（0：未知；1：博士；2：硕士；3：大学生；4：高中；5：初中；6：小学）
- **Query List**：用户历史一个月的搜索词列表

## 数据分析发现
- **标签分布**：
  - 年龄以"0-18岁"、"19-23岁"、"24-30岁"用户为主，极少数为"未知"或"51岁以上"。
  - 性别以男性略多于女性，少量未知。
  - 学历以"初中"、"高中"、"大学生"为主，博士和硕士极少，部分未知。
- **查询词高频特征**：
  - 年轻用户（0-18岁、19-23岁）高频词多为"学习"、"游戏"、"动漫"、"娱乐"等。
  - 成年用户（24-30岁、31-40岁）高频词涉及"求职"、"理财"、"结婚"、"房产"等。
  - 高学历用户（大学生及以上）关注"考研"、"留学"、"招聘"、"论文"等。
  - 男性用户高频词偏向"体育"、"科技"、"汽车"等，女性用户偏向"美妆"、"育儿"、"购物"等。
- **查询词多样性**：
  - 年轻用户和高学历用户的查询词更丰富，兴趣面更广。
  - 年龄较大或学历较低用户，查询词相对集中，兴趣面较窄。
- **兴趣偏好**：
  - 通过词云和高频词统计，可以直观看到各类用户的兴趣热点和行为特征。
  - 某些标签（如"未知"）下，查询词数量较少，分析价值有限。

## 主要分析流程
1. 数据读取与预处理
2. 不同标签（年龄、性别、学历）下的查询词高频统计与词云可视化
3. 查询词数量与多样性统计
4. 后续可扩展：特征工程、模型训练、分层抽样、交叉验证、模型部署等
