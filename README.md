# 基于NLP的搜索引擎用户画像项目

## 项目简介
本项目旨在利用自然语言处理和机器学习技术，基于用户历史查询词和人口属性标签，构建用户画像并预测新增用户的人口属性（年龄、性别、学历）。

## 数据说明
- **ID**：加密后的用户ID
- **Age**：年龄（0：未知；1：0-18岁；2：19-23岁；3：24-30岁；4：31-40岁；5：41-50岁；6：51-999岁）
- **Gender**：性别（0：未知；1：男性；2：女性）
- **Education**：学历（0：未知；1：博士；2：硕士；3：大学生；4：高中；5：初中；6：小学）
- **Query List**：用户历史一个月的搜索词列表

## 主要分析流程
1. 数据读取与预处理
2. 不同标签（年龄、性别、学历）下的查询词高频统计与词云可视化
3. 查询词数量与多样性统计
4. 后续可扩展：特征工程、模型训练、分层抽样、交叉验证、模型部署等

## 依赖环境
- Python 3.7+
- pandas
- matplotlib
- wordcloud

安装依赖：
```bash
pip install pandas matplotlib wordcloud
```

## 如何运行分析脚本
```bash
python3 scripts/data_analysis.py
```
